# Архитектурная документация сайта судебного участка № 121

## 1. Общие сведения
- **Название проекта:** Официальный сайт судебного участка № 121
- **Заказчик:** Судебный участок № 121 Люберецкого судебного района Московской области
- **Разработчик:** Калбодова Албина Амадбековна
- **Дата разработки:** 2026 год

## 2. Технологии
| Компонент | Технология |
|-----------|------------|
| Разметка | HTML5 |
| Стилизация | CSS3 |
| Шрифты | Google Fonts (Inter) |
| Иконки | Font Awesome |
| Хостинг | GitHub Pages |
| HTTPS | GitHub Pages (автоматически) |
| Контроль версий | Git, GitHub |
| Аналитика | GitHub Insights |

## 3. Соответствие 152-ФЗ
- На сайте размещено уведомление о согласии на обработку персональных данных
- Данные пользователей не собираются и не передаются третьим лицам

## 4. Адаптация под 1C-Bitrix
Для адаптации сайта под CMS 1C-Bitrix необходимо:
1. Перенести HTML-разметку в шаблон Bitrix
2. Заменить статический текст на динамические компоненты

## 5. Анализ эффективности
Используется встроенная статистика GitHub Insights.
